# Data Management System

## Description

This project is a simple data management system made with JavaScript.

The program uses:

- Objects
- Set
- Map
- Loops
- Data validation

The project works with products and shows information in the console.

---

## Features

- Create product data
- Validate product information
- Show product details
- Use Set methods
- Use Map methods
- Show object methods

---

- HTML5
- JavaScript
