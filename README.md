# Interactive Message System

## Description

This project is a simple interactive system made with JavaScript.

The program asks the user for:
- name
- age

Then the program validates the information and shows a message.

If the user is under 18, the program shows a message for minors.

If the user is 18 or older, the program shows a message for adults.

The program also validates:
- empty name
- empty age
- invalid numbers
- negative ages
